This chapter teaches how to adapt foundation models to specific use cases efficiently. You’ll explore full fine-tuning, parameter-efficient methods (LoRA, QLoRA, PEFT), supervised fine-tuning (SFT), preference optimization (RLHF, DPO, ORPO), and model merging techniques. Additional topics include hyperparameter optimization, quantization-aware training, distillation, and evaluation frameworks for fine-tuned models. By the end, you’ll understand the full spectrum of techniques used to create high-performing, domain-specific models while managing compute costs and avoiding catastrophic forgetting.
